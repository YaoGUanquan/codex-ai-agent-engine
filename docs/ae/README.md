# AE 工作流产物

这里存放使用 AI Agent Engine for Codex 时产生的过程文档。

## 目录说明

- `brainstorms/`：需求澄清和验收标准。
- `plans/`：实现计划。
- `reviews/`：代码或文档审查报告。
- `gates/`：验证和交付门禁证据。
- `handoffs/`：下次继续工作的交接说明。
- `experience/`：可复用的实现经验。
- `solutions/`：方案比较和选型记录。
- `archive/`：已完成或已废弃的过程记录。

## 与项目文档的关系

- `docs/ae` 保留为 AE 技能兼容的工作流产物目录。
- `docs/00-process` 记录执行中的方案、归档规则和可续跑状态。
- `docs/08-ai-memory` 记录跨会话复用的长期项目记忆。
