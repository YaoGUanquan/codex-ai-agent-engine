# PPTX Workflow

Use this flow for `.pptx` work:

1. Detect whether the deck already exists.
2. Decide whether the task is create, inspect, add content, update content, or validate.
3. Use OfficeCLI help to discover slide, shape, and render commands.
4. Prefer structured query or render output after modifications.
5. Use render or view verification before claiming slide layout success.

Common task classes:

- deck generation
- slide updates
- shape or text inspection
- render and validation
